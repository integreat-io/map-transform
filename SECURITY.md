# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.6.x   | :white_check_mark: |
| < 1.6   | :x:                |

## Reporting a Vulnerability

We welcome vulnerabilty reports. Please
[open an issue](https://github.com/integreat-io/map-transform/issues/new) and
describe what you've find.
